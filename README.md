# ShopZone E-Commerce SPA

## 🚀 Project Overview

ShopZone is a modern E-Commerce Single Page Application built with React, TypeScript, Tailwind CSS, React Router, and Context API.

This project was developed as part of Week 6 Internship Sprint focusing on:

- Client-side routing
- Global state management
- Context API
- Protected routes
- SPA architecture
- Responsive UI/UX
- API integration

## 📸 Features

### ✅ Routing
- Multi-page SPA using React Router
- Dynamic Product Routes
- Protected Checkout Route

### ✅ State Management
- Global Cart using Context API
- Persistent cart with localStorage
- Persistent authentication state

### ✅ Ecommerce Features
- Product listing from API
- Product details page
- Add to cart
- Quantity management
- Cart total calculation
- Search products

### ✅ UI/UX
- Responsive Design
- Skeleton loading
- Toast notifications
- Modern ecommerce UI
- Mobile responsive layout
